技能简介

通过 Tmux 技能，你可以让 AI 远程操控 tmux 会话：发送命令、收集输出、监控长时间运行的任务，适合自动化运维和交互式 CLI。

核心命令

# 启动命名会话
tmux new -s my-session

# 发送键位到指定窗格
/tmux send-keys -t my-session:0.1 "ls -la" Enter

# 抓取面板输出
/tmux capture-pane -t my-session:0 -p

# 等待任务完成
/tmux wait-for -S my-session:0

使用示例（自动化）

# 创建一个会话并运行 top
/tmux new -s monitor
/tmux send-keys -t monitor "htop" Enter

# 3秒后抓取结果
sleep 3
/tmux capture-pane -t monitor -p

注意事项

  • 目标机器需安装 tmux
  • 需要可访问的 tmux socket（默认 /tmp/tmux-UID）
  • 避免敏感命令泄露：捕获的 pane 内容不要存到公共位置
