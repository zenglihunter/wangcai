技能简介

GitHub 操作：Issues、PRs、CI 状态、日志查看。适合所有开发协作场景。

能力

  • 查看和筛选 Issues（按标签、分配、状态）
  • 检查 PR 状态、CI 跑测结果
  • 读取 Workflow 和 Job 日志
  • 创建和回复 Issue/PR（仅信息性，不进行代码修改）

使用场景

# 查看 open 状态的 issues
/github issues --state open --limit 10

# 检查某个 PR 的 CI 状态
/github pr checks 123

# 查看失败的工作流日志
/github workflow logs --run-id 456

限制

此技能为只读操作，不会执行代码合并、推送等写操作。复杂 Web UI 交互需使用浏览器工具。
