Weather

获取当前天气与天气预报，无 API key 需求

技能简介

Weather 技能使用 wttr.in 或 Open-Meteo 查询天气，不需要任何 API key，适合快速获取当前天气和未来几天的预报。

用法示例

# 查深圳当前天气
/weather 深圳

# 查北京未来 3 天预报
/weather 北京 --days 3

# 指定单位（公制/英制）
/weather 上海 --units metric

输出格式

支持 text、json、emoji 三种风格，默认简洁文本。数据来源：wttr.in（Free）和 Open-Meteo。

限制

  • 依赖第三方服务的可用性
  • 历史天气和灾害警报不在范围内
  • 仅获取公开气象数据，不涉及私密气象 API
