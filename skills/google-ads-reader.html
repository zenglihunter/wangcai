本技能通过 **Google Ads API** 的只读权限读取账户/活动/广告组/关键词的关键指标，生成简洁的文本/HTML 报告，便于每日快速了解投放表现和异常。

环境变量（全部需要）

  • GOOGLE_ADS_CLIENT_ID
  • GOOGLE_ADS_CLIENT_SECRET
  • GOOGLE_ADS_DEVELOPER_TOKEN
  • GOOGLE_ADS_LOGIN_CUSTOMER_ID （MCC 账号，无横杠）
  • GOOGLE_ADS_CLIENT_CUSTOMER_ID （具体子客户账号）
  • GOOGLE_ADS_REFRESH_TOKEN （只读 scope：https://www.googleapis.com/auth/adwords.readonly）

> 这些值都只保存在本机环境或 .env 文件中，绝不写入公开仓库。

使用方法

# 生成昨日日报（ amend 可指定日期，默认昨天）


python3 scripts/google_ads_report.py --date 2026-03-09 --output markdown


输出为 HTML（可用于博客嵌入）


python3 scripts/google_ads_report.py --output html

脚本会：

  • 拉取 Campaign、AdGroup、Keyword 级别的花费、点击、展示、转化等指标。
  • 计算 ROAS / CPA。
  • 对比前一日的环比变化，自动标红显著波动（>30%）。
  • 生成 Summary + Top N 高消耗关键词列表。

自动安排

可配合 OpenClaw cron 定时运行，例如每天 09:00 出前一天的日报：

0 9 * * * cd /home/user/.openclaw/workspace && /usr/bin/env python3 skills/google-ads-reader/scripts/google_ads_report.py --date yesterday --output markdown >> logs/ads-reader.log 2>&1

安全性

  • 本技能为**只读**，不会执行任何写入操作。
  • 所有 API 调用使用 OAuth refresh token，凭据不暴露。
  • 如需升级为可写版本（暂停/调价/增删），需重新授权并切换为 adwords 完整 scope。
