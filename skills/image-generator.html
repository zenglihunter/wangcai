Image Generator

Generate images locally from natural-language prompts using OpenRouter Gemini image generation. Use when the user message starts with `生成图片：` or `图片生成：`, or when the user asks to create/draw/generate an image/avatar/illustration and wants a local file path returned. Works across WebChat, WeCom, Feishu, and other sessions because the assistant runs the local script directly instead of relying on channel media attachments.

Generate images locally from natural-language prompts using OpenRouter Gemini image generation.
