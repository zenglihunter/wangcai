GitHub Issues

Fetch GitHub issues, spawn sub-agents to implement fixes and open PRs, then monitor and address PR review comments.

技能简介

一站式 GitHub Issues 管理：查问题、生成修复、提 PR、跟进 Review 评论。

用法示例

/gh-issues openai/openai-python --label bug --limit 5
/gh-issues openai/openai-python --watch --interval 5
/gh-issues my-org/my-repo --reviews-only --notify-channel -100123456789

核心能力

  • Fetch issues with filters (label, assignee, milestone, state)
  • Spawn ACP coding sessions to implement fixes
  • Auto-open PRs and monitor CI
  • Handle review comments by iterating
  • Flexible modes: run, session, cron, dry-run

配置

需要 GitHub CLI 认证：gh auth login。可选 --fork 指定 fork 账户，--notify-channel 发送通知到 Telegram/Discord。
