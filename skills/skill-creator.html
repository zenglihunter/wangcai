技能简介

Skill Creator 帮助你从零开始创建 OpenClaw AgentSkills：定义 SKILL.md、编写脚本、添加资源，并发布到 ClawHub 供他人安装。

工作流

# 1) 按交互式向导创建技能骨架
/skill-creator new my-skill

# 2) 编写 SKILL.md（描述、用法、参数）

# 3) 开发核心逻辑（scripts/*.py 或 *.mjs）

# 4) 测试
/skill-creator test ./my-skill

# 5) 发布到 ClawHub
/skill-creator publish ./my-skill

目录结构

my-skill/
├── SKILL.md           # 技能描述（markdown）
├── README.md          # 详细文档（可选）
├── scripts/           # 可执行脚本
│   └── run.py
├── lib/               # 共享模块（可选）
├── assets/            # 图片/图标
└── _meta.json         # 元数据（名称、版本）
